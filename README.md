# Steps to run

## Install NodeJS

https://nodejs.org/en

## Install VSCode

https://code.visualstudio.com/

## Install Git

https://git-scm.com/downloads

## Steps to run

#### Open repo in VS Code

#### Open Terminal in VS Code: control + ~

#### `git pull`
#### `npm install`
#### `npm start`

## Steps to stop

#### control + c
#### to kill 3000 port: `kill -9 $(lsof -ti:3000)`

## Steps to push changes

#### `git add .`
#### `git commit -m "commit message"`
#### `git push`

### `npm run build`
